# TUI Manager для P2P приложения

Современный текстовый интерфейс (TUI) для управления пиром в распределённой системе.

## Обзор

TUI менеджер предоставляет:
- 📡 **Обнаружение пиров** в сети через multicast/UDP
- 🔌 **Подключение к пирам** вручную по адресу IP:port
- 💬 **Чат с пирами** с историей сообщений и визуальным разделением входящих/исходящих
- 📊 **Просмотр активных сессий** с информацией о типе и статусе
- 📋 **Управление подключениями** - создание и закрытие сессий
- ℹ️ **Подробную информацию** о выбранных пирах и сессиях

## Архитектура

Модуль разделён на независимые слои для лёгкого изменения каждого аспекта:

```
internal/manager/tui/
├── tui_manager.go          # Основной менеджер, координация компонентов
├── config/
│   └── hotkeys.go          # Конфигурация горячих клавиш
├── styles/
│   ├── colors.go           # Цветовые темы
│   └── components.go       # Стили UI компонентов
├── errors/
│   └── handler.go          # Обработка и логирование ошибок
├── business/
│   ├── peer_ops.go         # Операции с пирами
│   ├── session_ops.go      # Операции с сессиями
│   └── chat_ops.go         # Операции чата
└── ui/
    ├── layout.go           # Макеты и основные компоненты
    ├── chat.go             # Окно чата
    └── status.go           # Строка состояния
```

### Слои

1. **Config** - конфигурация (горячие клавиши, настройки)
2. **Styles** - визуальное оформление (цвета, темы, стили компонентов)
3. **Errors** - обработка ошибок (типы ошибок, логирование, helper-функции)
4. **Business** - бизнес-логика (операции с пирами, сессиями, чатом)
5. **UI** - компоненты интерфейса (макеты, чат, строка состояния)
6. **Manager** - координация всех компонентов

## Горячие клавиши

### Основные команды

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Q` | Выход из приложения |
| `Ctrl+R` | Обновить списки пиров и сессий |
| `Ctrl+E` | Распространить информацию о себе (Emit) |
| `Ctrl+C` | Подключиться к выбранному пиру |
| `Ctrl+D` | Отключить выбранную сессию |
| `Ctrl+I` | Показать информацию о выбранном элементе |
| `Ctrl+H` | Показать справку по горячим клавишам |
| `Tab` | Переключение между панелями |
| `↑/↓` | Навигация по списку |
| `Enter` | Выбрать элемент |

### Чат

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Q` | Выйти из чата (сохраняя сессию) |
| `Ctrl+W` | Выйти из чата и закрыть сессию |
| `Enter` | Отправить сообщение |
| `Escape` | Назад / отменить действие |

## Использование

### Создание менеджера

```go
import (
    "github.com/first-debug/p2p/internal/manager/tui"
    peerstorage "github.com/first-debug/p2p/internal/storage/peer-storage"
    sessionstorage "github.com/first-debug/p2p/internal/storage/session-storage"
)

// Создаем хранилища
peerStore := peerstorage.NewMemoryPeerStorage()
sessionStore := sessionstorage.NewMemorySessionStorage()

// Создаем explorer для обнаружения пиров
explorer := explorer.NewUDPExplorer(cfg, peerInfo, peerStore)

// Создаем client для подключений
client := client.NewClient()

// Создаем TUI менеджер
manager, err := tui.NewTuiManager(explorer, peerStore, sessionStore, client)
if err != nil {
    log.Fatal(err)
}

// Запускаем менеджер (блокирующий вызов)
if err := manager.Run(); err != nil {
    log.Fatal(err)
}
```

### Остановка менеджера

```go
// Создаем контекст для остановки
ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
defer cancel()

// Останавливаем менеджер
manager.Stop(ctx)
```

## Настройка

### Изменение горячих клавиш

```go
import "github.com/first-debug/p2p/internal/manager/tui/config"

// Создаем кастомную конфигурацию
hotkeys := &config.HotkeyConfig{
    Quit:       tcell.KeyCtrlQ,
    Refresh:    tcell.KeyCtrlR,
    Connect:    tcell.KeyCtrlC,
    // ... остальные клавиши
}

// Передаем в менеджер (требуется модификация NewTuiManager)
```

### Изменение темы

```go
import "github.com/first-debug/p2p/internal/manager/tui/styles"

// Тёмная тема (по умолчанию)
theme := styles.ModernDarkTheme()

// Светлая тема
theme := styles.ModernLightTheme()

// Или создайте свою тему
theme := &styles.ColorTheme{
    Background:     tcell.ColorDefault,
    Foreground:     tcell.ColorWhite,
    // ... остальные цвета
}
```

## Взаимодействие с другими модулями

```
┌─────────────────┐
│   TuiManager    │
└────────┬────────┘
         │
    ┌────┴────┬────────────┬──────────────┬──────────┐
    │         │            │              │          │
    ▼         ▼            ▼              ▼          ▼
┌────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│Explorer│ │PeerStorage│ │SessionStorage│ │Client   │ │Server   │
└────────┘ └──────────┘ └──────────┘ └──────────┘ └──────────┘
```

- **Explorer** - обнаружение пиров в сети
- **PeerStorage** - хранилище информации о пирах
- **SessionStorage** - хранилище активных сессий
- **Client** - создание подключений к пирам
- **Server** - обработка входящих подключений

## Расширение функциональности

### Добавление новых горячих клавиш

Отредактируйте `config/hotkeys.go`:

```go
func DefaultHotkeyConfig() *HotkeyConfig {
    return &HotkeyConfig{
        // ... существующие клавиши
        NewAction: tcell.KeyCtrlN, // Новая горячая клавиша
    }
}
```

### Добавление новых стилей

Отредактируйте `styles/colors.go`:

```go
func ModernDarkTheme() *ColorTheme {
    return &ColorTheme{
        // ... существующие цвета
        NewColor: tcell.ColorPurple, // Новый цвет
    }
}
```

### Добавление новых UI компонентов

Создайте файл в `ui/` и добавьте компонент в `UIComponents`:

```go
// ui/new_component.go
package ui

import "github.com/rivo/tview"

type NewComponent struct {
    view *tview.TextView
}

func NewNewComponent() *NewComponent {
    return &NewComponent{
        view: tview.NewTextView(),
    }
}
```

## Отладка

Логирование ошибок осуществляется через стандартный логгер с префиксом `[TUI Errors]`.

Для включения отладочного вывода установите переменную окружения:

```bash
export DEBUG=true
```

## Примеры

### Подключение к пиру

1. Выберите пир в левом списке (↑/↓)
2. Нажмите `Ctrl+C` для подключения
3. Откроется чат с выбранным пиром

### Отправка сообщения

1. Выберите сессию в правом списке или подключитесь к пиру
2. Введите сообщение в поле ввода (нижняя часть правой панели)
3. Нажмите `Enter` для отправки

### Закрытие сессии

1. Выберите сессию в списке
2. Нажмите `Ctrl+D` для закрытия сессии
3. Или `Ctrl+W` в окне чата для закрытия сессии и выхода из чата
